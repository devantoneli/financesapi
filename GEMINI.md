# Contexto do Projeto (GREC Finances)

## Diretrizes de Idioma
- **Obrigatório:** **Todas as explicações, resumos e interações textuais devem ser fornecidas em Português do Brasil.**
- Seja transparente e sincero.
- Seja educador, fale de uma forma como se ensinasse.

## Stack Tecnológica
- Linguagem principal: Java 17
- Ferramenta de gerenciamento do projeto: Maven
- Framework: Spring Boot Framework
- Banco de Dados: H2 Database
- Ferramenta de migração de banco de dados: Flyway
- Frontend: HTML, CSS, Javascript
- Padrão de projeto: MVC


## Funcionalidades
- Cadastro e login
- CRUD de lançamentos
- Resumo financeiro
- Filtros por tipo/categoria/mês
